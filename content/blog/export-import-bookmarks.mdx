The following instructions demonstrate how to export bookmarks from and import bookmarks to any web browser in Windows. This should be used in instances where switching computing environments (i.e. from Citrix to bare-metal Windows) does not carry over bookmarks in any given browser.

---

## Firefox

### Export

1. Select the hamburger menu (3 stacked horizontal lines) at the top right of the browser and then select Bookmarks from the list.

<Image
  src="/_static/blog/blog1/Firfox_bookmarks_export_step1.png"
  width="718"
  height="404"
  alt="Firefox Bookmark Export Step 1"
/>

2. Select Manage Bookmarks at the very bottom of the list to access the bookmark Library screen.

<Image
  src="/_static/blog/blog1/Firfox_bookmarks_export_step2.png"
  width="718"
  height="404"
  alt="Firefox Bookmark Export Step 2"
/>


3. In the Library bookmarks screen, select Import and Backup from the top menu, then select Backup... from the list.

<Image
  src="/_static/blog/blog1/Firfox_bookmarks_export_step3.png"
  width="718"
  height="404"
  alt="Firefox Bookmark Export Step 3"
/>


4. Save the file to your desktop. This file will be available on your desktop in both Citrix and a bare-metal Windows environment.




### Import

1. Select the hamburger menu (3 stacked horizontal lines) at the top right of the browser and then select Bookmarks from the list.
2. Select Manage Bookmarks at the very bottom of the list to access the bookmark Library screen.
3. In the Library bookmarks screen, select Import and Backup from the top menu, then select Restore then Choose FIle.

<Image
  src="/_static/blog/blog1/Firfox_bookmarks_import_step1.png"
  width="718"
  height="404"
  alt="Firfox Bookmark Import Step 1"
/>

4. Choose the file you saved to the desktop previously. Restoring from a backup will replace all the bookmarks in the browser with the bookmarks in the backup.

## Chrome

### Export

1. Select the kebab menu (3 vertical dots) at the top right of the browser. Select Bookmarks then Bookmark Manager.

<Image
  src="/_static/blog/blog1/Chrome_bookmarks_export_step1.png"
  width="718"
  height="404"
  alt="Chrome Bookmark Export Step 1"
/>

2. In the Bookmarks tab that opens, select the kebab (3 vertical dots) to the right of the Search Bookmarks field then select Export Bookmarks.

<Image
  src="/_static/blog/blog1/Chrome_bookmarks_export_step2.png"
  width="718"
  height="404"
  alt="Chrome Bookmark Export Step 2"
/>

3. Save the file to your desktop. This file will be available on your desktop in both Citrix and a bare-metal Windows environment.

### Import

1. Select the kebab menu (3 vertical dots) at the top right of the browser. Select Bookmarks then Bookmark Manager.
2. In the Bookmarks tab that opens, select the kebab (3 vertical dots) to the right of the Search Bookmarks field then select Import Bookmarks.
3. Choose the file you saved to the desktop previously.
4. You bookmarks will be in a folder called Imported. You can proceed to organize these bookmarks by dragging and dropping them.

<Image
  src="/_static/blog/blog1/Chrome_bookmarks_import_step1.png"
  width="718"
  height="404"
  alt="Chrome Bookmark Import Step 1"
/>

## Microsoft Edge


### Export

1. Select the star hamburger menu (3 horizontal lines on a star) at the top right of the browser.
2. Select the meatball menu (3 horizontal dots) at the top right of the Favorites menu.
3. Select Export Favorites.

<Image
  src="/_static/blog/blog1/Edge_bookmarks_export_step1.png"
  width="718"
  height="404"
  alt="Edge Bookmark Export Step 1"
/>

4. Save the file to your desktop. This file will be available on your desktop in both Citrix and a bare-metal Windows environment.

### Import

1. Select the star hamburger menu (3 horizontal lines on a star) at the top right of the browser.
2. Select the meatball menu (3 horizontal dots) at the top right of the Favorites menu.
3. Select Import Favorites.
4. On the settings page that opens, in the Import browser data now section, select Choose what to import.

<Image
  src="/_static/blog/blog1/Edge_bookmarks_import_step1.png"
  width="718"
  height="404"
  alt="Edge Bookmark Import Step 1"
/>

5. Under Import form list select Favorites of bookmarks HTML file then select Choose file

<Image
  src="/_static/blog/blog1/Edge_bookmarks_import_step2.png"
  width="718"
  height="404"
  alt="Edge Bookmark Import Step 2"
/>

6. Choose the file you saved to the desktop previously.
